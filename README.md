# i3-wm config 

| Item         | Programs   |
|--------------|------------|
| WM           | i3-wm      |
| Bar          | i3status   |

| Shortcut        | Programs                         |
|-----------------|----------------------------------|
| Super + Enter   | Run a terminal                   |
| Alt + Shirt     | Change keyboard layout           |
| Super + d       | Run a program launcher           |

To know all the default i3-wm keyboard shortcuts, visit the [i3 man page](https://i3wm.org/docs/refcard.html)
